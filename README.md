## Structure
The main compositions is divided in 3 items

1. Frontend projects
	* Angular project
	* Razor + MVC Project
	* Blazor + MVC Project
	* Vanilla Project
	* Mobile Project (Xamarin or MAUI)
	
2. Middleware projects
	* Restful API
	* SOAP message
	
3. Backend project
	* .NET 5.0 or superior
	
4. Database
	* SQL Server
	* Postgre SQL (as secondary)
